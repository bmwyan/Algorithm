一、递归四路：
1）定义递归终结条件
2）处理当前层逻辑
3）下探到下一层
4）清理当前层：可能会有全局变量等
思维要点：
1.不要人肉递归
2.找到最近最简单的方法，将其拆解成成可重复解决的问题（重复子问题）
3.数据归纳法思维
JAVA代码模板：
public void recur(int level,int param)
{
    if(level>MAX_LEVEL){
        //process result
        return;
    }

    process(level,param);

    recur(level+1,newParam);

    //restore current status

}
